# 🌈 Personería Estudiantil — Valeria Rolong
### Colegio Cruzada Social

> **"Brillando juntas, transformamos nuestro futuro"**

Sitio web de campaña para la Personería Estudiantil, construido con **Vite + TypeScript** y ambientación temática de *My Little Pony*.

---

## 🚀 Cómo correr el proyecto

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Vista previa del build
npm run preview
```

## 📁 Estructura

```
├── index.html          → Página de inicio
├── discurso.html       → Discurso de campaña
├── propuestas.html     → Propuestas
├── objetivos.html      → Objetivos general y específicos
├── sugerencias.html    → Formulario de sugerencias y QR
├── src/
│   ├── main.ts         → Lógica TypeScript
│   └── style.css       → Estilos MLP
├── images/
│   ├── logo.png        → Logo de la candidata (agregar)
│   └── qr-sugerencias.png → Código QR (agregar)
└── vite.config.ts
```

## 🖼️ Agregar imágenes

- **Logo**: guarda la imagen como `images/logo.png`
- **QR**: genera el QR en [qr-code-generator.com](https://qr-code-generator.com) y guárdalo como `images/qr-sugerencias.png`
